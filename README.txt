QuadraticEquation's class must get 3 variables of double type, but first one can not be equal to 0.

if you want to connect to your database and get coefficients for QuadradicEquation you should enter your MYSQL user in "username" string on line 7;

also you should enter your MYSQL password in "password" string on line 8;

lastly you should enter database name, were you have your coefficients in "DBName" string on line 9.
